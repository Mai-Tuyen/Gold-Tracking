# 🪙 Gold Track

Welcome to **Gold Track** — a modern web app for tracking gold prices in Vietnam.

This is where you can view live prices by brand (SJC, DOJI, PNJ, etc.), explore charts and history, and (with an account) create and manage price alerts with optional push notifications.

## 👋 About

Gold Track is a full-stack PWA built with Next.js. It focuses on real-time price display, clear charts, and optional user accounts for saving alert rules and receiving push notifications when your thresholds are hit.

## 🚀 Tech Stack

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Supabase** (auth & database)
- **TanStack Query**
- **Framer Motion**
- **Radix UI**
- **Zustand**
- **next-intl**
- **Chart.js**
- **Serwist** (PWA)

## 📂 Features

- Real-time gold prices by brand (SJC, DOJI, PNJ, etc.)
- Price charts and history
- Price alerts (threshold and volatility)
- Google authentication
- Create and manage your own alerts
- Push notifications when alerts trigger
- Responsive layout & PWA support
- Dark mode
